#### Memoization(메모하기) 기법!

중복된 재귀호출을 피하는 방법이다.

~~~c
{
 	if(f[n] != 0) return f[n];
    else{
        if(n=1 || n=2)	f[n] = 1;  // f[n]값이 0 이면 fib(n)가 한번도 수행되지 않음을 의미
        else{
			f[n] = fib(n-1)+fib(n-2);
        }
        return f[n];
        
    }
}
~~~
