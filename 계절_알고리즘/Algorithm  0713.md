### Algorithm  0713

##### 알고리즘 설계와 분석의 기초

<hr>

#### 바람직한 알고리즘 

1. 명확해야 한다 - 이해하기 쉽고 간명
2. 효율적 이어야 한다.



#### 알고리즘 수행 시간

log(n) < n < nlog(n) < n^2 < n^3 < 2^n

*nlog(n) 만 해도 좋은 알고리즘!  n^2 도 비교적 좋은 알고리즘 이라고 할 수 있다.*



#### 재귀와 귀납적 사고

- 재귀 = 자기호출(recursion)

  > 재귀적 구조란? 어떤 문제 안에 크기만 다를 뿐 성격이 똑같은 작은 문제들이 포함되어 있는 것이다.
